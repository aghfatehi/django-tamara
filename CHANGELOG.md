# Changelog

## v1.0.0 (2024-01-01)

- Initial release
- Full Online Checkout flow
- Payment Types API
- Order management (Get, Authorise, Capture, Cancel, Refund)
- Webhook management (Register, List, Get, Update, Delete)
- In-store checkout support
- Configurable URL prefix and middleware
- Multi-country and multi-currency support
- Transaction logging model
- Django 4.2, 5.0 & 5.1 support
- Python 3.9+ support
